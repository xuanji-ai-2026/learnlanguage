# 汉越语学习工具 v2.0

## 项目简介

汉越语学习工具是移动端语言学习应用，帮助用户学习汉语和越南语。

## 技术栈

### 后端
- Python 3.11+
- FastAPI
- SQLite/PostgreSQL

### 前端
- React Native (Expo)
- TypeScript

### 部署
- Docker
- Docker Compose

## 项目结构

- backend/ - 后端代码
- frontend/ - 前端代码
- docs/ - 文档
- scripts/ - 脚本

## 开发指南

### 后端开发

1. 创建新的词汇文件
2. 更新API接口
3. 运行测试

### 前端开发
